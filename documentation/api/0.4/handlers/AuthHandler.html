This namespace provides methods to authenticate with the Red Hat Network.

Namespace: auth

Method: checkAuthToken

Description:
This method is used to see if an external service is handing back an authorized token indicating that Spacewalk trusts the requester in some manner. This is currently used in the cobbler integration.

Parameters:

  • string login - login of user to check against token
  • string token - token to validate

Returns:

  • int - 1 if the token is valid with this username, 0 otherwise.
